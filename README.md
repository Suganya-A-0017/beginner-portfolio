##  Portfolio Website (HTML & CSS)

This is a simple and responsive **personal portfolio website** built using **HTML and CSS**. The project focuses on creating a clean UI and showcasing basic front-end development skills.

###  Features

*  **Home Page**

  * Introduction section
  * Clean layout and design
  * Responsive structure

*  **Contact Page**

  * Simple contact form UI
  * User-friendly design

###  Technologies Used

* HTML5
* CSS3

### Purpose

This project was created as a beginner-level portfolio to practice:

* Web page structuring using HTML
* Styling and layout using CSS
* Building a basic multi-page website
